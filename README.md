# Kotlin Reaction Game 🎮

## Overview

This project is a grid-based reaction game built using Kotlin and Java Swing.
The objective is to quickly identify and click the green square before time runs out.
Each round increases the grid size, making the game progressively more difficult.

---

## Features

* Dynamic grid that increases in size each round
* Randomized target square (green button)
* Countdown timer system
* Game over screen with restart functionality
* Interactive GUI using Java Swing

---

## Technologies Used

* Kotlin
* Java Swing (JFrame, JPanel, JButton)
* Event-driven programming

---

## How It Works

1. The game starts with a small grid of buttons
2. One button is randomly colored green
3. The player must click the green button to advance
4. Each correct click increases the grid size (difficulty)
5. Clicking the wrong square or running out of time ends the game
